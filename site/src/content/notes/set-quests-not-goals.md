This comes from this really great piece: [https://www.raptitude.com/2024/08/do-quests-not-goals/](https://www.raptitude.com/2024/08/do-quests-not-goals/)

Instead of setting goals, you should go on quests.

A goal is something that you are meant to accomplish at some point. A quest is something you start today.

A quest is an adventure that you set off on. A quest changes you.

When you set a goal, you're disappointed if you don't achieve it. But when you begin a quest, you acknowledge that the pursuit of the something is the important part. The process, not the product.
